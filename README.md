# 🎮 Jogo da Adivinhação

Um jogo interativo de adivinhação desenvolvido com foco em lógica de programação web e estilização moderna. O computador gera um número secreto aleatório e o jogador tem um limite de tentativas para acertar, recebendo dicas dinâmicas a cada palpite.

---

## 🚀 Funcionalidades

* **Geração Aleatória:** O sistema define um número secreto entre 1 e 100 a cada partida.
* **Sistema de Tentativas:** O jogador possui no máximo 4 chances para adivinhar.
* **Contador Dinâmico:** Mostra em tempo real quantas chances ainda restam na tela.
* **Dicas em Tempo Real:** Informa se o número secreto é MAIOR ou MENOR que o chute atual.
* **Validação de Entrada:** Protege o jogo contra números vazios ou fora do intervalo (1 a 100).
* **Interface Responsiva:** Design centralizado e otimizado para navegação.

---

## 🎨 Visual e Estilização

O design do projeto foi construído do zero com CSS e adota uma estética inspirada no estilo *Cyberpunk*:
* Fundo escuro com degradê linear (`#000` para `#62005a`).
* Elementos estruturados em caixas com bordas arredondadas e sombras neon integradas (`box-shadow`).
* Tipografia limpa baseada na fonte *Roboto*.
* Botões interativos com efeitos de transição suave ao passar o mouse (`hover`).

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação dos elementos da página, inputs e parágrafos de retorno.
* **CSS3:** Alinhamentos inteligentes, Flexbox, gradientes de fundo e efeitos visuais.
* **JavaScript (ES6):** Manipulação da árvore do DOM, controle de estruturas condicionais (`if/else`) e operadores lógicos.

---

## 📦 Como Rodar o Projeto

1. Faça o clone ou baixe este repositório.
2. Certifique-se de que os arquivos `index.html`, `cssjogo.css` e `jogojs.js` estejam na mesma pasta.
